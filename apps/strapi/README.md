## Node.js

Tested with Node.js v16, you can use [NVM](https://github.com/nvm-sh/nvm) to switch between different Node.js versions:

```bash
nvm install 16
nvm use 16
```

### Build

```bash
cd strapi
npm run build
```

### Start

```bash
cd strapi
npm run start
```
